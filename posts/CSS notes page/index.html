简介

层叠样式表(英文全称：Cascading Style Sheets)是一种用来表现HTML（标准通用标记语言的一个应用）或XML（标准通用标记语言的一个子集）等文件样式的计算机语言。CSS不仅可以静态地修饰网页，还可以配合各种脚本语言动态地对网页各元素进行格式化。

特性

  • 层叠性： 相同选择器给设置相同的样式，此时一个样式会覆盖（层叠)另一个 冲突的样式。原则：就近原则（样式冲突） 不冲突就不层叠
  • 继承性： 子标签会继承父标签的某些样式。（text-，font-，line-这些元素开头的可以继承，以及color属性）与文字相关
  • 优先级： 当同一个元素指定多个选择器，选择器相同时，则执行层叠性，选择器不同时，则根据选择器权重执行
  • 权重的叠加： 复合选择器会有权重叠加的问题，权重会叠加，但不会有进位

CSS 笔记

    --- CSS 属性 ---

    (一) 盒模型属性
  • width 宽度
  • height 高度
  • border 边框
  • margin 外边距定位
  • padding 内边距，留白
  • background 背景
  • background-color 背景颜色
  • background-image 背景图像
  • float 浮动
  • display 生成盒的（模式）方式
  • position（top right bottom left） 位置定位
  • overflow 溢出属性
  • box-sizing 盒子大小
    • (二) 字体文本属性
  • color 字体颜色
  • font-family 字体
  • font-size 字号
  • font-weight 字体加粗
  • text-align 水平对齐方式
  • line-height 行高
  • text-indent 首行缩进
    • (三) 超链接及列表属性
  • text-decoration 文本修饰
  • list-style 列表样式

    • --- 随笔记录 ---

  • 文本溢出处理
    单行:
    white-space: nowrap;   /* 禁止换行 */
    overflow: hidden;   /* 溢出隐藏 */
    text-overflow: ellipsis;   /* 文本溢出处理 */
    多行:
    display: -webkit-box;   /* 禁止换行 */
    -webkit-line-clamp: 3;   /* 文本显示行数 */
    -webkit-box-orient: vertical;   /* 文本排列方向 */
    text-overflow: ellipsis;   /* 文本溢出处理 */
    ( 存在兼容性问题 )
    或
    height: 90px;
    line-height: 30px;
    overflow: hidden;
    ( 不存在兼容性问题, 但不会有省略号 )
  • 像素阴影:
    filter: drop-shadow( x, y, 模糊度, 颜色 );
  • 模糊背景:
    backdrop-filter: blur();
